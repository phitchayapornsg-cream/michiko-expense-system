# Michiko Expense App

## Use locally

Open `index.html` in Google Chrome or Microsoft Edge. The app works without an internet connection.

## Publish a link for other people

This is a static website. Upload the complete contents of this folder to a static hosting service, such as Netlify Drop, Cloudflare Pages, GitHub Pages, or a company web server. The hosting service must use `index.html` as the start page.

Keep the `assets` folder beside `index.html`, `styles.css`, and `app.js`. Do not rename or move these files after upload.

## Important data note

The current app stores records in each browser's Local Storage. People opening the same web link can use the app, but they do not automatically see or share each other's records. For shared company data, add a database and user sign-in before publishing for general team use.
